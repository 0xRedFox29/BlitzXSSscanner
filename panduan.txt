LANGKAH-LANGKAH PENGGUNAAN BLITZ XSS SCANNER

1. Unduh file blitzxssscanner.py atau copy kode program
2. Pastikan memiliki text editor seperti VS Code (rekomendasi)
3. Mengunduh ChromeDriver dan python 3.11 di microsoft store 
4. run kode program 
5. ketik "pip install selenium colorama" di terminal (tanpa tanda kutip)
6. tunggu sampai selesai
7. running tools dengan nama python namafile.py

jika ingin menjalankan di CMD atau powershell bisa sesuaikan dengan file path
contoh: 
 
C:/Users/Spart/OneDrive/Desktop/XSSscaner/python XSSscanpy.py

tampilan di CMD

C:/Users/Spart>cd OneDrive
C:/Users/Spart/OneDrive>cd Desktop
C:/Users/Spart/OneDrive/Desktop>cd XSSscaner
C:/Users/Spart/OneDrive/Desktop/XSSscaner>python XSSscanpy.py
